Databases

A database is a collection of information or data that is stored electronically in a computer.

Databases are controlled by a Database Management System (DBMS), which allows users to add, organize, and manipulate stored data.

Image from vecteezy.com


Here are a few different types of databases:

Relational Databases: Organize data into tables with rows and columns. Use SQL for querying and managing data.

Example: MySQL

NoSQL Databases: Designed to handle unstructured data.

Example: MongoDB

Object-Oriented Databases: Store data in the form of objects.
