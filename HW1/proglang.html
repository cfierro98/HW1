Programming Languages is a system of notation used to write computer programs.

It allows for humans to communicate with computers and control its behaviors.

Image from iStockPhoto.com

Key components of a programming language include:

Syntax: Specifies how characters are arranged to form valid statements and expressions.

print("Hello, world!")

Data Types: Values stored and manipulated in a program such as integers, strings, booleans, etc.


name = "Claudia"   #string
age = 28           #integer
isStudent = true   #boolean

Control Structures: Statements that control the flow of a program.


if num % 2 == 0:
    print("Even")
else:
    print("Odd")
